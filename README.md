# 💱 Converte Já

## 📱 O que é?

**Converte Já** é um aplicativo mobile que permite converter valores entre diferentes moedas de forma rápida e simples. Ideal para viagens, compras internacionais ou acompanhar o câmbio do dia a dia.

## 🎯 Funcionalidades

- Conversão entre [Real, Dólar, Euro, etc.]
- Taxas de câmbio [atualizadas em tempo real / fixas]
- Interface limpa e intuitiva
- Resultado instantâneo
